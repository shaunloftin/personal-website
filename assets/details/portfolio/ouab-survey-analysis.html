OUAB Survey Analysis

Project information

  • Category: Coding
  • Client: Ohio Union Activities Board
  • Project date: Spring 2018 - Fall 2020
  • Project URL: https://github.com/shaunloftin/survey-analysis

OUAB, or the Ohio Union Activities Board, is the Ohio State University's campus programming board. OUAB plans hundreds of events for students throughout the school year.

To better gauge event interest among Ohio State students, I designed and published multiple surveys to be distributed. These surveys received thousands of responses each semester.

This simple Java program helps clean raw survey data which would be later visualized using Tableau. Findings from these surveys were compiled into internal reports delivered to Ohio State administrators.
