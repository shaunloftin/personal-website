Montaña de Luz - Website Redesign

Project information

  • Category: Web design
  • Client: Montaña de Luz
  • Project date: Summer 2020
  • Project URL: https://www.montanadeluz.org/
  • Skills used: HTML, CSS, JS, PHP, WordPress

Montaña de Luz is a Columbus non-profit dedicated to empowering children, youth, and families affected by HIV/AIDS in Honduras.

When looking to make changes to an outdated site, I proposed and implemented design recommendations using WordPress with custom code additions. I also minimized back-end risk through backups, updates, and minimizing SQL vulnerabilities.
