The Happiness Campaign Website Design

Project information

  • Category: Web Design
  • Client: The Happiness Campaign
  • Project date: Winter 2020
  • Project URL: https://thehappinesscampaign.org/
  • Skills used: HTML, CSS, JS, PHP, WordPress

The Happiness Campaign is an Ohio State University student organization focused on mental health awarness and establishing healthy, positive habits.

I was contracted to build a website from scratch where participants can log a daily task, participate in research surveys, and view collective campaign projects. I started the website using a custom WordPress Template, designed the database schema, added PHP user interactions, and implemented version control.
